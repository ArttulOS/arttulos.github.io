The Path Forward: ArttulOS Roadmap

Discover our development journey, upcoming milestones, and how we're building the future of ArttulOS, together.

Current Focus: Q1-Q2 2025

Core System Stability

Refining the base system derived from Rocky Linux, ensuring maximum stability and compatibility. Extensive testing of kernel modules and core utilities.

Status: In Progress | Target: Q2 2025

Desktop Environment v1

Initial release of the ArttulOS themed desktop environment. Focus on usability and unique visual identity.

Status: In Progress | Target: Q2 2025

Package Repository Setup

Establishing and populating the initial ArttulOS package repositories with essential applications and development tools.

Status: Planned | Target: Q1 2025

Next Up: Q3-Q4 2025

Alpha/Beta Release Program

Launching public alpha and beta testing phases to gather community feedback and identify bugs.

Status: Planned | Target: Q3 2025

Initial Documentation

Developing core documentation: installation guides, user manuals, and basic troubleshooting.

Status: Planned | Target: Q3 2025

Security Hardening

Implementing enhanced security features, default configurations, and security update infrastructure.

Status: Planned | Target: Q4 2025

Future Goals: 2026 & Beyond

ArttulOS 1.0 Stable Release

The first official stable release of ArttulOS, ready for general use, incorporating community feedback.

Status: Planned | Target: Early 2026

Expanded Hardware Support

Ongoing efforts to ensure compatibility with a wider range of modern and legacy hardware components.

Status: Ongoing

Community & Dev Tooling

Developing enhanced tools and platforms for community contributions, such as automated build services or translation portals.

Status: Exploratory

Our roadmap is a living document, shaped by progress and invaluable community input.
